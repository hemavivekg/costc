What is possible option?
Assume if there is only one match in a series (for the teams A & B), there will be only two possible options.
Possible Options
1 2
Match 1 Winner A B


Assume if there is two matches in a series, there will be only four possible options. 22 = 4
Possible Options
1 2 3 4
Match 1 Winner A A B B
Match 2 Winner B A B A


Assume if there is three matches in a series, there will be only eight possible options. 23 = 8
Possible Options
1 2 3 4 5 6 7 8
Match 1 Winner A A A A B B B B
Match 2 Winner A B A B A A B B
Match 3 Winner A A B B A B A B
